# Movies
This is a React app that shows a list of movies using The Movie Database (TMDb) API. Click on a movie to see more details about it, including the overview, release date, and rating. The site is responsive for both mobile, tablets and desktop.

Practicing of using React & React Router.

API Integration: Fetched data from TMDb using fetch and stored it in state with useState.

Responsive Design:
Movie list uses CSS Grid
Movie detail page uses Flexbox, with the image stacked on mobile and side-by-side on desktop

Extra Features:
Rating shows one decimal
Loading state while fetching data

Error handling if a movie is not found
